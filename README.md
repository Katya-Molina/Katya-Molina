# About Me 🌟

🎓 **Engineering Student:**
Currently pursuing a degree in **Computer Engineering** at the **Central American University José Simeón Cañas (UCA)**.

💻 **Growing Frontend Developer:**
Passionate about web development with experience in **JavaScript**, **React**, **Vue.js**, **Tailwind CSS**, and **Ant Design**.

📊 **Business Intelligence & Data:**
Con experiencia práctica desarrollando soluciones de datos de extremo a extremo: desde la ingesta hasta el reporte final.
- Desarrollo de **pipelines de datos y procesos ETL/ELT** para integrar múltiples fuentes.
- Transformación y análisis de datos con **SQL**, **Microsoft Fabric** y **Azure Synapse Analytics**.
- Creación de **dashboards y reportes de BI** para monitoreo operativo y toma de decisiones.
- Modelado de datos en **Data Warehouse** y **Lakehouse** para análisis de negocio.

🚀 **Agile Methodologies:**
Knowledgeable in **Scrum** with experience using tools like **Jira** for efficient project management.

🌱 **Lifelong Learner:**
Always looking for new opportunities to learn and grow in **fullstack web development** and **technology project management**.
